# Bank-Loan-Report-Power-BI-Analytics-Dashboard
Bank Loan Report analyzing 38,576 loan applications (2021) — 3-page Power BI dashboard tracking $435.7M funded, 86.18% good loan rate, and 13.82% charge-off risk. Covers MTD/MoM KPIs, grade-wise risk profiling (A–G), regional analysis, and borrower segmentation. All DAX measures validated against SQL queries.
